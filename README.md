# BomberAPP

Una aplicación de entrenamiento para oposiciones de bomberos en Valencia, España.

## Características

- **Calculadora de Baremos**: Calcula tus puntuaciones en tiempo real para las 6 pruebas físicas basadas en el Decreto 163/2019.
- **Seguimiento de Progreso**: Visualiza tus estadísticas actuales en las pruebas principales.
- **Objetivos Activos**: Configura y sigue tus metas de examen.
- **WOD Diario**: Recibe entrenamientos diarios personalizados.

## Pruebas Soportadas

1. **Cuerda**: Escalada de cuerda (6m hombres, 5m mujeres)
2. **3000m**: Carrera de resistencia
3. **60m**: Carrera de velocidad
4. **Natación**: 100m de natación
5. **Press Banca**: Levantamiento de pesas (50kg hombres, 30kg mujeres)
6. **Torre**: Subida a torre (Consorcio y Ayuntamiento de Valencia)

## Instalación

1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Inicia el servidor de desarrollo: `npm run dev`

## Uso

- Selecciona tu género para cálculos precisos
- Ingresa tus tiempos/resultados en la calculadora
- Revisa tu estado de aptitud para cada prueba
- Sigue tu progreso en el dashboard principal

## Tecnologías

- React
- TypeScript

## Contribución

Este proyecto está basado en el Decreto 163/2019 de la Comunidad Valenciana. Para actualizaciones o correcciones de baremos, consulta la legislación oficial.
